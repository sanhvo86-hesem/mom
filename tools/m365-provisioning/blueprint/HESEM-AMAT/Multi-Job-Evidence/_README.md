# Multi-Job-Evidence for AMAT

When a CAPA root cause / 8D field-failure / banned-substance heat-lot / audit
finding spans MULTIPLE Jobs of AMAT parts, the parent record lives here.
Per-Job NCR/CAPA in Jobs/.../Quality back-links via NCR-Master-List metadata.

Examples for AMAT:
- CAPA from supplier defect affecting 5 AMAT Jobs
- 8D field-failure spanning 3 prior AMAT shipments
- Banned-substance heat-lot trace across 7 AMAT Jobs
- Audit finding referencing 5 AMAT Jobs
