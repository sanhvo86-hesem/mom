# Multi-Job-Evidence for TEL

When a CAPA root cause / 8D field-failure / banned-substance heat-lot / audit
finding spans MULTIPLE Jobs of TEL parts, the parent record lives here.
Per-Job NCR/CAPA in Jobs/.../Quality back-links via NCR-Master-List metadata.

Examples for TEL:
- CAPA from supplier defect affecting 5 TEL Jobs
- 8D field-failure spanning 3 prior TEL shipments
- Banned-substance heat-lot trace across 7 TEL Jobs
- Audit finding referencing 5 TEL Jobs
