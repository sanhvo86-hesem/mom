# Multi-Job-Evidence for ASML

When a CAPA root cause / 8D field-failure / banned-substance heat-lot / audit
finding spans MULTIPLE Jobs of ASML parts, the parent record lives here.
Per-Job NCR/CAPA in Jobs/.../Quality back-links via NCR-Master-List metadata.

Examples for ASML:
- CAPA from supplier defect affecting 5 ASML Jobs
- 8D field-failure spanning 3 prior ASML shipments
- Banned-substance heat-lot trace across 7 ASML Jobs
- Audit finding referencing 5 ASML Jobs
