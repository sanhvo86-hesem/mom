# Multi-Job-Evidence for LAM

When a CAPA root cause / 8D field-failure / banned-substance heat-lot / audit
finding spans MULTIPLE Jobs of LAM parts, the parent record lives here.
Per-Job NCR/CAPA in Jobs/.../Quality back-links via NCR-Master-List metadata.

Examples for LAM:
- CAPA from supplier defect affecting 5 LAM Jobs
- 8D field-failure spanning 3 prior LAM shipments
- Banned-substance heat-lot trace across 7 LAM Jobs
- Audit finding referencing 5 LAM Jobs
