Reference implementation

Apache (.htaccess hiện hành)

# 04-Bieu-Mau/.htaccess Options -Indexes <IfModule mod_mime.c> AddType application/vnd.openxmlformats-officedocument.spreadsheetml.sheet .xlsx AddType application/vnd.ms-excel.sheet.macroEnabled.12 .xlsm AddType application/vnd.ms-excel .xls </IfModule> <IfModule mod_headers.c> <FilesMatch "\.(xlsx|xlsm|xls)$"> Header set Content-Disposition "attachment" Header set X-Content-Type-Options "nosniff" </FilesMatch> </IfModule>

Nginx reference

location /04-Bieu-Mau/ { types { application/vnd.openxmlformats-officedocument.spreadsheetml.sheet xlsx; application/vnd.ms-excel.sheet.macroEnabled.12 xlsm; application/vnd.ms-excel xls; } if ($request_uri ~* \.(xlsx|xlsm|xls)$) { add_header Content-Disposition "attachment" always; add_header X-Content-Type-Options "nosniff" always; } }

IIS reference

Bật staticContent cho xlsx/xlsm/xls và thêm custom response headers: Content-Disposition: attachment X-Content-Type-Options: nosniff
