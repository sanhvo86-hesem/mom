| Control Area | UoM Evidence |
|---|---|
| Electronic records | MEASVAL envelope, UoM rule records, manifest records, audit events |
| Audit trail | Trace id, before/after payloads, event contracts, audit hash |
| Signature manifestation | signer id, timestamp, signature meaning, manifest hash |
| Signer identity | active human actor and RBAC permission checks |
