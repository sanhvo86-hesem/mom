## Đổi gì (file-based module CRUD, backward-compatible)
`mom/api/controllers/ModuleSchemaController.php` + `mom/api/routes/platform-routes.php`:
1. **Version history** — `saveSchema` snapshot prior revision vào `data/modules/_versions/<id>/vNNNNNN.json` (giữ 30 bản) TRƯỚC khi ghi đè → save thôi "ghi đè mù", last-write-wins thành recoverable. `module_schema_versions` (list) + `module_schema_restore_version` (rollback, tạo forward version mới, không rewrite history).
2. **Soft-delete/tombstone/restore** — `deleteSchema` mặc định set `status='deleted'` (giữ file) thay HARD `@unlink`; `purge:true` mới hard-delete (vẫn snapshot trước). `module_schema_restore` lift tombstone. `listSchemas` ẩn deleted (trừ `&includeDeleted=1`).
3. **Concurrency optimistic version-lock** — `saveSchema` opt-in `baseVersion`: nếu ≠ version trên đĩa → **409 `version_conflict`** kèm `currentVersion`+`current`, KHÔNG ghi đè. version nay monotonic từ authority on-disk (không từ payload stale).
4. **Binding data_contract validate** — `bindingReport()` walk `dataSource.api` refs, phân loại theo endpoint-catalog; trả non-fatal report trong save response + action riêng `module_schema_validate_bindings` → thay silent console.warn.
