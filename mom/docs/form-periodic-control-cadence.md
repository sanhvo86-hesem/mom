# HESEM QMS — Periodic Control Cadence Baseline V0

## Daily
- Triage lỗi tải workbook / path / auth.

## Weekly
- Smoke-check 15 form trọng yếu từ portal và nội dung tài liệu.

## Monthly
- Checksum spot-check tối thiểu 10 workbook.
- Header verification ở server thật.
- Link audit changed scope.

## Quarterly
- Full critical-form UAT pack.
- Review desktop endpoint baseline.
- Review NOT_ISSUED_IN_V0 codes không bị lộ như active.

## On-change
- Recompute checksum.
- Update registry / docs_custom / descriptions.
- Impact review.
- Targeted UAT.
