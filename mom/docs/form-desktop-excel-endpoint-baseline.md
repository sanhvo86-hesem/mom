# HESEM QMS — Desktop Excel Endpoint Baseline V0

## Nguyên tắc
- Người dùng chỉ tải workbook active từ thư viện FRM có kiểm soát.
- Không dùng file rename local hoặc file gửi qua chat/email như bản chính thức.
- Desktop Excel là ứng dụng chuẩn để mở workbook active.
- Trusted location, nếu áp dụng, phải do IT công bố và quản trị tập trung.
- Trusted publisher chỉ dùng cho macro project đã được phê duyệt và ký số.
- Không cho phép người dùng tự tạo thói quen “Enable Content cho mọi file”.

## Kiểm tra tối thiểu trên pilot endpoints
1. `.xlsx` mở mặc định bằng Microsoft Excel desktop.
2. Download folder đúng quy ước IT.
3. Người dùng biết nơi lưu record department-approved.
4. Người dùng biết cách xử lý Protected View / file from Internet theo hướng dẫn IT.
5. Không có uncontrolled sync folder được dùng như nơi phát hành chính thức.
