# HESEM QMS — Windows Desktop Excel Endpoint Baseline V0

- `.xlsx` mở bằng Microsoft Excel desktop.
- Downloads chỉ là điểm nhận file tạm, không phải nơi lưu record chính thức.
- Người dùng lưu record/evidence vào đường dẫn bộ phận đã phê duyệt.
- Không tự thêm Trusted Location hoặc Trusted Publisher ngoài rule do IT quản lý.
- Không đổi tên file local rồi phát hành lại như bản chính thức.
