CFG gói hàng + chữ ký điện tử/ack + SnapshotID CP3 — Hồ sơ công việc khung sườn PPL+FRONTLINE_LEADS Tạo Hồ sơ công việc theo template trước khi chạy; bắt buộc có index pack tối thiểu. Thư mục đúng tên + cấu trúc + file index CP4 — Nhận hàng cert cổng kiểm soát D-WHS/QC/QA Không cấp vật tư/outsource cho sản xuất khi cert thiếu; cert gắn Job/PO/lot. Bộ chứng từ chứng nhận + nhận hàng acceptance + link PO/lot CP5 — Kiểm tra báo cáo tính đầy đủ QC+QA Báo cáo phải có key + Rev + điều kiện đo; thiếu trường = TẠM GIỮ. IR/CMM/FAI pack + xác nhận phê duyệt + dấu thời gian CP6 — Bộ hồ sơ giao hàng cổng kiểm soát D-WHS+QA bộ hồ sơ giao hàng theo PackID; bảng kiểm "đủ pack bắt buộc"; khai báo hồ sơ trước ship. Bộ hồ sơ giao hàng + CoC + bảng kiểm + hồ sơ trạng thái CP7 — Hết hiệu lực cách ly / biệt trữ QA/QMS Thu hồi bản hết hiệu lực; kho hết hiệu lực chỉ đọc; PHẢI loại khỏi Point-of-use tại phân xưởng. Danh mục hết hiệu lực + log thay đổi + thông báo phát hành CP8 — Truy cập rà soát SYSTEM_OWNERS + FRONTLINE_LEADS Review quyền theo vai trò; tách quyền đọc/ghi theo Khu vực; khóa chia sẻ ngoài. Biên bản rà soát quyền truy cập + danh sách nhóm truy cập
Đánh dấu mức "bằng chứng-ready": hồ sơ đã khai báo (đọc-chỉ / duy nhất) + có index + có key + có người chịu trách nhiệm. Thiếu 01 yếu tố = FAIL cổng kiểm soát.
