Lessons learned · điền sau họp (Lean Change cycle)

Insights

Điền sáng 2026-06-01 bởi QMS_ENG + QA_MGR. Ghi 3 phát hiện từ W03: issue categories, retrieval outliers, standup attendance và any mismatch giữa old system với QMS docs.

Options

Điền sáng 2026-06-01 bởi QMS_ENG + QA_MGR. Nêu ít nhất 3 lựa chọn cho W04: keep pilot scope, add wrong-revision drill only, extend pilot or prepare conditional Wave 2.

Experiments

Điền sáng 2026-06-01 bởi QMS_ENG + QA_MGR. Mỗi experiment phải có hypothesis, metric, owner role, sample size và deadline trước 2026-06-06.

Không dùng Lessons Learned để sửa thầm SOP-605, WI-201 hoặc DRL-E2E. Nếu experiment cho thấy tài liệu cần đổi, QMS_MGR mở DCR hoặc issue có traceability theo WI-105/WI-106 và giữ source-of-truth discipline.
