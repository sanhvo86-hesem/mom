Power Automate Flow Design (16 Lists)

  • Điều kiện kích hoạt source: SharePoint file create/modify event OR Webhook từ MOM ERP (NCR creation in Epicor).
  • Idempotency key: NcrId / CapaId / FaiId / etc. — flow re-run safe.
  • Failure handler: Queue retry 3 lần, alert ticket vào Action-Tracker-Master.
  • Lag SLA: 5 phút từ Tệp save → list update.
  • Tệp-move flow (Job-Closure): Move Job-Dossier → Archive, rewrite Canonical-Đường dẫn column trên tất cả Lists liên kếted, set Lifecycle-Trạng thái=Archived, emit đánh giá_events row.
