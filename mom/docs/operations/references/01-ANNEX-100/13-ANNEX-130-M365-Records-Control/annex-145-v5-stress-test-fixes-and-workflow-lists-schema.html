Power Automate luồng Design (16 Lists)

  • Trigger nguồn: SharePoint file create/modify event hoặc Webhook từ MOM ERP (NCR creation in Epicor).
  • Idempotency key: NcrId / CapaId / FaiId / etc. — luồng re-run safe.
  • Failure handler: Queue retry 3 lần, alert ticket vào Action-Tracker-Master.
  • Lag SLA: 5 phút từ file save → danh sách update.
  • File-move luồng (Job-Closure): Move Job-Dossier → Archive, rewrite Canonical-Path column trên tất cả Lists linked, set Lifecycle-State=Archived, emit audit_events row.
