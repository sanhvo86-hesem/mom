v7 Unified Workspace + CustomerCode-Keyed Kiến trúc

BỐI CẢNH — v6 thất bại vì over-segmentation:
v6 tách 4 khách hàng-OEM không gian riêng (HESEM-AMAT, -LAM, -ASML, -TEL) — tưởng là perfect IB nhưng phản hồi người dùng: "không cần HESEM-LAM, HESEM-ASML gì cả, tất cả được quản lý bên trong mã khách hàng". Đây là cách international contract CNC manufacturer (Moog Aerospace, Heico, Onto Innovation, tầng-1 contract shops) thực sự làm — 1 workspace duy nhất với CustomerCode làm key thư mục. Thêm khách mới = thêm thư mục (không cần tạo không gian mới, provision IB segment mới, đào tạo member mới).
v7 = Industry Standard Pattern cho contract CNC manufacturer scale 200-500 employees:
  1. 3 không gians duy nhất: HESEM-Workspace (transactional), HESEM-Internal (PII/IT/OT/Pháp lý/Exec), HESEM-Archive (long-term lưu giữ).
  2. CustomerCode là top-level thư mục trong Workspace/Customers/{Code}/. Permission via thư mục inheritance + SG-Cust-{Code} groups.
  3. Industry vocabulary: Drawings, BOM, CAM, Inspection, Specification, FAI, ECN, Planning, Purchasing, Manufacturing, chất lượng, Giao hàngping, Reports.
  4. MOM = SSOT cho controlled docs (Manual/chính sách/SOP/WI/RACI/ANNEX/đào tạo/JD/biểu mẫu). M365 chỉ chứa transactional bằng chứng.
  5. IB ở thư mục level (không không gian) — đủ mạnh cho contract CM thông thường, đơn giản admin.
  6. Adding new khách hàng = create thư mục, không cần tạo không gian/IB segment mới.

7. Adding a New khách hàng (v7 advantage)

tình huống: HESEM win new khách hàng "KLA Tencor" (KLA). Cần workspace cho KLA.

v6 (6-không gian) effort:
  1. Create new SharePoint không gian HESEM-KLA
  2. Set up M365 group SG-Cust-KLA-Members
  3. Configure Information Barrier segment Customer-KLA + pairwise IB rules với AMAT/LAM/ASML/TEL
  4. Configure không gian mặc định sensitivity label
  5. Configure không gian lưu giữ chính sách
  6. Provision thư mục template (Parts/, Jobs/, POs/, etc.)
  7. Add không gian to hub navigation
  8. Train members
Estimate: ~3-5 ngày người + IT admin overhead

v7 (3-không gian) effort:
  1. Tạo thư mục HESEM-Workspace/Customers/KLA/ bằng PowerShell script (copy từ _TEMPLATE-NewCustomer/)
  2. Create AAD group SG-Cust-KLA-Members
  3. Apply thư mục permission (inherit từ Customers/, override với SG-Cust-KLA-Members + IB tag Customer-KLA)
  4. Đã xong
Estimate: ~2-4 giờ
Scaling advantage: v7 linear scaling. v6 non-linear (không gian count grows, IB ma trận grows quadratic O(N²) pairs).
