v7 Unified Workspace + CustomerCode-Keyed Architecture

BỐI CẢNH — v6 thất bại vì over-segmentation:
v6 tách 4 customer-OEM site riêng (HESEM-AMAT, -LAM, -ASML, -TEL) — tưởng là perfect IB nhưng phản hồi người dùng: "không cần HESEM-LAM, HESEM-ASML gì cả, tất cả được quản lý bên trong mã khách hàng". Đây là cách international contract CNC manufacturer (Moog Aerospace, Heico, Onto Innovation, tier-1 contract shops) thực sự làm — 1 workspace duy nhất với CustomerCode làm key folder. Thêm khách mới = thêm folder (không cần tạo site mới, provision IB segment mới, training member mới).
v7 = Industry Standard Pattern cho contract CNC manufacturer scale 200-500 employees:
  1. 3 sites duy nhất: HESEM-Workspace (transactional), HESEM-Internal (PII/IT/OT/Legal/Exec), HESEM-Archive (long-term retention).
  2. CustomerCode là top-level folder trong Workspace/Customers/{Code}/. Permission via folder inheritance + SG-Cust-{Code} groups.
  3. Industry vocabulary: Drawings, BOM, CAM, Inspection, Specification, FAI, ECN, Planning, Purchasing, Manufacturing, Quality, Shipping, Reports.
  4. MOM = SSOT cho controlled docs (Manual/chính sách/SOP/WI/RACI/ANNEX/Training/JD/Forms). M365 chỉ chứa transactional evidence.
  5. IB ở folder level (không site) — đủ mạnh cho contract CM thông thường, đơn giản admin.
  6. Adding new customer = create folder, không cần tạo site/IB segment mới.

7. Adding a New Customer (v7 advantage)

Scenario: HESEM win new customer "KLA Tencor" (KLA). Cần workspace cho KLA.

v6 (6-site) effort:
  1. Create new SharePoint site HESEM-KLA
  2. Set up M365 group SG-Cust-KLA-Members
  3. Configure Information Barrier segment Customer-KLA + pairwise IB rules với AMAT/LAM/ASML/TEL
  4. Configure site default sensitivity label
  5. Configure site retention chính sách
  6. Provision folder template (Parts/, Jobs/, POs/, etc.)
  7. Add site to hub navigation
  8. Train members
Estimate: ~3-5 ngày người + IT admin overhead

v7 (3-site) effort:
  1. Tạo folder HESEM-Workspace/Customers/KLA/ bằng PowerShell script (copy từ _TEMPLATE-NewCustomer/)
  2. Create AAD group SG-Cust-KLA-Members
  3. Apply folder permission (inherit từ Customers/, override với SG-Cust-KLA-Members + IB tag Customer-KLA)
  4. Đã xong
Estimate: ~2-4 giờ
Scaling advantage: v7 linear scaling. v6 non-linear (site count grows, IB matrix grows quadratic O(N²) pairs).
