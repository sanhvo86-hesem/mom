6 Smart Scroll UX

4 quy tac scroll bat buoc. Muc tieu: chi data di chuyen, layout co dinh, scrollbar khong chiem dien tich.

RULE 1: CHI DATA SCROLLHeader, KPI, filter, footer: LUON STICKY. Chi main/sidebar zones scroll.
RULE 2: SCROLLBAR AN MAC DINHscrollbar-width:none → hover: scrollbar-width:thin. Overlay, khong chiem dien tich.
RULE 3: RE CHUOT RA DE SCROLL TRANGKhi tong zones > viewport, re chuot ra khoi data area de scroll layout. Data area van chi scroll data.
RULE 4: SEARCH/FILTER LUON HIENThanh tim kiem, loc luon co dinh phia tren data area. User khong phai scroll de tim kiem.

12 Governance — Quy tac bat buoc

10 Quy tac BAT BUOC
  1. Moi module phai co templateId — khong ngoai le.
  2. Khong tu tao layout rieng — dung zone system.
  3. Block chi vao zone cho phep — allowedBlocks enforced.
  4. Token override chi o template level — module KHONG override.
  5. Scroll tuan theo zone config — sticky zone KHONG scroll.
  6. Template thay doi = module thay doi — khong unlink.
  7. Custom template phai co toi thieu 3 zones — header, main, footer.
  8. Preset goc KHONG xoa — chi clone + chinh sua.
  9. Search/filter luon sticky — user khong scroll de tim.
  10. Scrollbar overlay — khong chiem dien tich noi dung.
10 Anti-patterns — KHONG LAM
  1. Module tu dinh nghia grid layout rieng
  2. Hardcode spacing/color trong module JS
  3. Scroll toan trang thay vi data-only scroll
  4. Tao module khong chon template
  5. Override token tai module instance level
  6. Dat block sai zone (vd: form vao kpi-bar)
  7. Xoa preset goc
  8. Unlink module khoi template
  9. An search/filter bar khi scroll
  10. Scrollbar chiem dien tich, day noi dung
