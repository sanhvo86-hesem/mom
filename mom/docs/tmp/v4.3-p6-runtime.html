6.1Template → DOM Rendering Pipeline
Pseudocode: 1. Lấy templateId từ module config hoặc key hesem_active_template_{moduleId}; fallback sang binding hiện tại hesem_module_template_binding. 2. Resolve template → layout config { zones[], gridTemplate, density, blocks[] } từ server hoặc hesem_layout_templates. 3. Tạo DOM shell: <div class="tpl-shell" data-template="{id}" data-density="{density}">. 4. Apply template-level token overrides (Layer 3) vào pending token map, chưa inject CSS variables. 5. Cho mỗi zone trong zones[]: a. Validate zone.type thuộc 8 zone types của Section 33. b. Tạo <section class="zone zone-{type}" data-zone="{type}" role="{ariaRole}">. c. Apply zone-level token overrides (Layer 4) vào zone token map. d. Cho mỗi block trong zone.blocks[]: i. Validate block.type nằm trong zone.allowedBlocks. ii. Validate props bằng JSON Schema từ Section 29. iii. Lazy load block module nếu registry chưa có factory. iv. Mount block: block.mount(containerEl, props, context). v. Subscribe block to data source và template event bus. 6. Resolve cascaded tokens: System CSS → Admin config → Template → Zone → User preference. 7. Inject computed CSS variables vào tpl-shell và từng zone element. 8. Attach ResizeObserver, IntersectionObserver và ErrorBoundary cho từng block. 9. Mark template ready, emit runtime:template-ready, đo template switch time.

6.2Block Lifecycle Hooks
interface BlockLifecycle { onMount(container: HTMLElement, props: BlockProps, context: AppContext): void; onUpdate(changedProps: Partial<BlockProps>): void; onTokenChange(tokens: Record<string, string>): void; onResize(width: number, height: number): void; onData(data: unknown, meta: DataMeta): void; onSuspend(): void; onResume(): void; onDestroy(): void; onError(error: Error): FallbackUI | null; }
HookKhi gọiRule bắt buộcVí dụ block
onMountSau khi container và context sẵn sàng.Không fetch trùng nếu data source đã preloaded; phải đăng ký cleanup handles.data-table tạo header và skeleton rows.
onUpdateProps hoặc binding đổi nhưng block vẫn cùng type.Diff props, không remount toàn bộ nếu chỉ đổi filter/sort/density.chart-line đổi date range.
onTokenChangeCascade token thay đổi ở admin/template/zone/user.Chỉ đọc token đã computed; không tự ghi localStorage trong block.kpi-row đổi spacing và font size.
onResizeZone hoặc viewport resize.Debounce bằng requestAnimationFrame; chart không đo layout liên tục.gantt-chart recalc timeline width.
onDataData source trả dữ liệu hoặc realtime delta.Apply delta idempotent; giữ selection nếu rowKey còn tồn tại.alarm-list nhận alarm mới.
onSuspendBlock ra khỏi viewport hoặc tab inactive.Dừng polling, animation và heavy observer; giữ state nhẹ.digital-twin-view pause scene.
onResumeBlock vào viewport lại.Request delta từ timestamp cuối, không full reload nếu chưa quá TTL.sensor-card refresh latest value.
onDestroyTemplate switch, block remove hoặc route leave.Unsubscribe event bus, abort fetch, remove timers, disconnect observers.data-table remove row listeners.
onErrorFactory/mount/update/data render throw.Trả fallback UI nếu tự xử lý được; nếu null, ErrorBoundary render fallback chuẩn 6.9.chart-bar lỗi config series.

bundleSizeRecommendedBudget gzip per block, target dưới 50KB.Build warning nếu vượt budget; release gate fail nếu critical path vượt.
