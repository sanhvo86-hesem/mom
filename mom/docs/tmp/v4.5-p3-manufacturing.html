??OEE Calculation Contract

OEE trong HESEM MOM phải tính từ raw fields thống nhất để Production, Quality, Finance và Executive dashboard không lệch số khi dùng cùng một shift hoặc work order.

Formula definition
OEE = Availability x Performance x Quality

Availability = Run Time / Planned Production Time
  Planned Production Time = Shift Duration - Planned Breaks
  Run Time = Planned Production Time - Unplanned Downtime

Performance = (Ideal Cycle Time x Total Pieces) / Run Time
Quality = Good Pieces / Total Pieces
  Good Pieces = Total Pieces - Reject Pieces

??Preventive Maintenance Workflow

PM workflow kết nối lịch bảo trì, work order maintenance, checklist kỹ thuật và KPI MTBF/MTTR để giảm downtime không kế hoạch.

PM work order form
  • Machine ID, PM Type: daily, weekly, monthly, annual.
  • Checklist items với pass/fail/comment và evidence attachment.
  • Parts used, time spent, next PM date, technician signature.
  • PM task tự động tạo WO type Maintenance khi đến hạn.
MTBF / MTTR KPI display
MTBF = Total Operating Time / Number of Failures
MTTR = Total Repair Time / Number of Repairs
Display bằng KPI cards có trend sparkline, drill-down theo machine, line, failure mode và technician.
