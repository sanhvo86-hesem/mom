5.2Roving Tabindex Patterns
ROVING TABINDEX RULE Mỗi composite widget chỉ có 1 tab stop. Item active dùng tabindex="0", item còn lại dùng tabindex="-1". Arrow keys di chuyển focus nội bộ; Tab thoát khỏi widget.
Toolbar pattern
WidgetRolesKeyboard modelFocus return
Action toolbarrole="toolbar", child buttonsTab vào active button; Left/Right chuyển item; Home/End nhảy đầu/cuối; Enter/Space kích hoạt.Sau modal/action đóng, focus quay về button đã kích hoạt.
 
<div role="toolbar" aria-label="Thao tác phiếu CAPA">
  <button tabindex="0">Lưu</button>
  <button tabindex="-1">Gửi duyệt</button>
  <button tabindex="-1">Xuất PDF</button>
</div>

function onToolbarKeydown(event, items, index) {
  if (event.key === 'ArrowRight') moveFocus(items, index + 1);
  if (event.key === 'ArrowLeft') moveFocus(items, index - 1);
  if (event.key === 'Home') moveFocus(items, 0);
  if (event.key === 'End') moveFocus(items, items.length - 1);
}

function onTreeKeydown(event, node) {
  if (event.key === 'ArrowDown') focusNextVisibleNode();
  if (event.key === 'ArrowUp') focusPreviousVisibleNode();
  if (event.key === 'ArrowRight') expandOrFocusFirstChild(node);
  if (event.key === 'ArrowLeft') collapseOrFocusParent(node);
}

function onGridKeydown(event, cell) {

5.3Vietnamese Typography Requirements
FONT STACK ĐÃ VALIDATE Dùng -apple-system, 'Segoe UI', 'Noto Sans', 'Roboto', Arial, Helvetica, sans-serif. Stack này tương thích tiếng Việt có dấu trên macOS, Windows, Android, iOS và browser kiosk phổ biến.
OVERLINE WARNING Không dùng overline 10px cho tiếng Việt có dấu. Minimum cho Vietnamese label có dấu là 11px, line-height 1.4, letter-spacing 0 đến 0.04em tùy uppercase ngắn.

5.5Reading Order & CSS Reordering Rule
ABSOLUTE RULE KHÔNG BAO GIỜ dùng order, flex-direction: row-reverse, grid-auto-flow:dense hoặc grid placement để tạo visual order khác DOM order. Screen readers đọc DOM order. Ngoại lệ duy nhất là RTL layout transforms, hiện chưa áp dụng cho HESEM MOM.
Case Allowed Forbidden Why Fix
