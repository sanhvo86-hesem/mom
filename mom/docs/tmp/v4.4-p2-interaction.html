??Pattern 12 — Inline Edit Interaction Pattern

Bổ sung quy chuẩn inline edit cho bảng dữ liệu, form rút gọn và các field có thể chỉnh nhanh. Pattern này mở rộng Section 31, phải khớp state matrix của Section 4, form validation ở Section 21, table behavior ở Section 22, modal conflict ở Section 23, keyboard model ở Section 26 và toast feedback ở Section 30.

DOInline edit PHẢI có keyboard trigger. KHÔNG chỉ dựa vào double-click. Cell focus được bằng Tab, Enter mở edit mode, Escape hủy và Tab di chuyển sang cell editable tiếp theo.
WARNINGInline edit KHÔNG áp dụng cho trường regulated, trường e-signature, trường đã khóa hoặc field cần reason code theo Section 15.
