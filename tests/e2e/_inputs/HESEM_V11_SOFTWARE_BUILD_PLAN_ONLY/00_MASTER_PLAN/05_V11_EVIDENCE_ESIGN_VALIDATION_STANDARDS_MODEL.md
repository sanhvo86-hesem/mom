## Control rule

A standard reference is not enough. It must become artifact, owner, gate, evidence, test/review method, rollback/restriction, frontend/API/workflow control, audit event, signature meaning where applicable and stop-rule.

## Validation posture

M04 supplies validation planning, not site-specific validation evidence. Site/tenant intended-use, risk classification, protocol, execution evidence, deviation handling and summary report remain required before regulated use.
